# 🏥 heart_deseas_project Prediction System

## 📌 Project Overview

This project is a Machine Learning-based web application that predicts whether a healthcare insurance claim will be **Approved ✅ or Rejected ❌** based on patient and claim details.

The model is trained using historical healthcare claims data and deployed using Streamlit for real-time predictions.

---

## 🎯 Objective

To build a predictive system that helps insurance companies automate claim approval decisions using machine learning.

---

## 🧠 Technologies Used

* Python 🐍
* Pandas & NumPy
* Scikit-learn
* Matplotlib & Seaborn
* Streamlit (for deployment)
* Joblib (model saving)

---

## ⚙️ How It Works

1. Data preprocessing (handling missing values, encoding)
2. Feature selection and transformation
3. Model training using Random Forest Classifier
4. Model evaluation
5. Deployment using Streamlit

## 🚀 Installation & Setup

### Step 1: Clone the repository

```
git clone https://github.com/your-username/healthcare-claim-prediction.git

```

### Step 2: Install dependencies

```
pip install -r requirements.txt
```

### Step 3: Run the application

```
streamlit run app.py
```
